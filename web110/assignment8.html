Five SQL Examples...

This is an example of the SQL SELECT statement:

  • SELECT * FROM Customers WHERE Country = 'Mexico';

This is an example of the SQL WHERE clause with USA as the filter:

  • SELECT * FROM Customers WHERE Country = 'USA';

This is an example of SQL COUNT() function:

  • SELECT COUNT(CustomerID) FROM Customers;

Throughout this lesson, while playing around with the SQL exercises on W3 Schools' website, I learned that SQL statements is mainly pluging and playing- with changing the order a bit. For example, from the get-go, you're provided with a syntax like "SELECT COUNT(CustomerID) FROM Customers;" for SQL Count() function. All you need to do, depending on what values you need, is plug in lets say the ProductID after SELECT COUNT and Products after FROM to see how many products there are. This rule applies with all other SQL statements and only require adding in a value.
